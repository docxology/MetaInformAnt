# Sequence Analysis Agents

The Sequence module handles fundamental DNA sequence operations.

## Capabilities
- **Manipulation**: Reverse complement, translation, transcription.
- **Analysis**: GC content, molecular weight, ORF finding.
- **Pattern Matching**: Motif finding, consensus generation.
- **Enzymes**: Restriction site analysis.

## Key Files
- `core.py`: Basic sequence object and operations.
- `composition.py`: Nucleotide composition analysis.
- `motifs.py`: Motif discovery.
- `restriction.py`: Restriction enzyme analysis.
