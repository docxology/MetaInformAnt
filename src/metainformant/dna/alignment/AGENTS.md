# Alignment Agents

The Alignment module handles pairwise and multiple sequence alignment.

## Capabilities
- **Pairwise Alignment**: Needleman-Wunsch (global) and Smith-Waterman (local).
- **MSA**: Multiple sequence alignment (wrappers for MUSCLE/MAFFT).
- **Distances**: Evolutionary distance calculation (p-distance, Jukes-Cantor).

## Key Files
- `pairwise.py`: Pairwise alignment algorithms.
- `msa.py`: Multiple sequence alignment tools.
- `distances.py`: Genetic distance matrices.
