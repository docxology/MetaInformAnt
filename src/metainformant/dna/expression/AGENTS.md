# Expression Agents

The Expression module models the Central Dogma processes.

## Capabilities
- **Transcription**: DNA -> RNA conversion.
- **Translation**: RNA -> Protein conversion using codon tables.
- **Codon Analysis**: Codon Usage Bias (CUB), CAI, RSCU.

## Key Files
- `transcription.py`: Transcription logic.
- `translation.py`: Translation logic.
- `codon.py`: Codon usage statistics.
