# Agent Directives: examples/visualization

## Role
Example scripts demonstrating visualization capabilities.

## Contents
Examples for:
- Basic plots (line, scatter, histogram)
- Genomic visualizations (Manhattan, QQ)
- Heatmaps and expression plots
- Network visualizations
- Phylogenetic trees
- Statistical plots
- Animations

## Key Patterns
- All plots return matplotlib Figure objects
- Output saved to `output/visualization/`
- Consistent styling across examples
