## Example Categories

### Sequence Processing Examples
- **Basic Operations**: Reading FASTA files, reverse complement, GC content calculation
- **Quality Control**: Sequence validation, N-content analysis, length distributions
- **Manipulation**: Subsequence extraction, sequence concatenation, format conversion
- **Statistics**: K-mer frequency analysis, complexity measures, entropy calculations

### Alignment and Comparison Examples
- **Pairwise Alignment**: Global/local alignment with different scoring matrices
- **Multiple Alignment**: Progressive alignment and consensus generation
- **Distance Calculations**: Evolutionary distances (Jukes-Cantor, Kimura)
- **Similarity Assessment**: Identity percentages and conservation scoring

### Phylogenetic Analysis Examples
- **Tree Construction**: Neighbor-joining and UPGMA tree building
- **Bootstrap Support**: Confidence estimation through resampling
- **Tree Visualization**: ASCII and graphical tree representations
- **K-mer Phylogeny**: Fast phylogenetic inference from sequence composition

### Population Genetics Examples
- **Diversity Metrics**: Nucleotide diversity (π), segregating sites, Watterson's θ
- **Neutrality Tests**: Tajima's D, Fu & Li's D*/F*, Fay & Wu's H
- **Genetic Differentiation**: F_ST calculations between populations
- **Hardy-Weinberg**: Allele frequency and equilibrium testing

### Composition and Structure Examples
- **GC Analysis**: Genome-wide GC content and GC skew calculations
- **Codon Usage**: RSCU, CAI, and codon preference analysis
- **Motif Discovery**: IUPAC pattern matching and PWM construction
- **Restriction Mapping**: Enzyme site identification and virtual digestion

### Genomic Data Integration Examples
- **NCBI Downloads**: Genome assembly retrieval and metadata extraction
- **Entrez Queries**: Database searching and record parsing
- **Variant Analysis**: VCF file processing and SNP identification
- **Annotation Integration**: GFF/GTF file parsing and feature extraction
