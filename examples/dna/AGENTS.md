# Agent Directives: examples/dna

## Role
Example scripts demonstrating DNA sequence analysis.

## Contents
- `example_sequences.py` - Sequence processing (GC content, k-mers)
- `example_alignment.py` - Sequence alignment algorithms
- `example_population.py` - Population genetics statistics
- `example_phylogeny.py` - Phylogenetic tree construction

## Key Demonstrations
- FASTA/FASTQ parsing
- Global and local alignment
- Tajima's D, nucleotide diversity
- Neighbor-joining trees
