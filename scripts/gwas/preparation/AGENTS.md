# Agent Directives: scripts/gwas/preparation

## Role
Data preparation scripts for GWAS workflows.

## Contents
- `download_genome_scale_data.sh` - Download reference genomes and data
- `download_honeybee_variants.py` - Download A. mellifera variants
- `generate_phenotypes.py` - Generate phenotype files
- `generate_synthetic_variants.py` - Create synthetic test data
- `query_bioproject_metadata.py` - Query SRA/BioProject metadata

## Purpose
Prepare input data for GWAS analysis:
- Download and organize variant data
- Generate phenotype matrices
- Create synthetic data for testing
