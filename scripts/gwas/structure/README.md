# STRUCTURE

## Overview
Command-line helpers for GWAS Structure workflows. Scripts should remain thin wrappers around `src/metainformant/` implementations and be run from the repository root with `uv`.

## Contents
- [run_kinship.py](run_kinship.py)
- [run_pca.py](run_pca.py)

## Usage
```bash
uv run python scripts/gwas/structure/run_kinship.py --help
```
