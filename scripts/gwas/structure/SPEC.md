# SPEC: GWAS Population Structure

Technical specification for population structure analysis scripts.

> Inherits from [scripts/gwas/SPEC.md](../SPEC.md).

## Purpose

Scripts for calculating PCA and kinship matrices to control for population stratification.

## Key Methods

| Method | Description |
|--------|-------------|
| PCA | Principal Component Analysis for genetic ancestry |
| Kinship Matrix | Relatedness estimation (VanRaden method) |
