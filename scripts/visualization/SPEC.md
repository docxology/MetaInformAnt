# SPEC: Visualization Scripts

Orchestration of high-quality plotting and animation pipelines.

## Workflows

- `generate_dashboard.py`: Creates a multi-panel summary of analysis results.
- `run_animation_pipeline.py`: Generates MP4/GIF animations from time-series biological data.

## Standards

- **Output Formats**: Prioritize SVG and PDF for publication-quality results.
- **Styling**: All plots must adhere to the project's aesthetic standards (dark mode options, Inter typography).
