# PHENOTYPE

## Overview
Command-line helpers for Phenotype workflows. Scripts should remain thin wrappers around `src/metainformant/` implementations and be run from the repository root with `uv`.

## Contents
- [load_antwiki_example.py](load_antwiki_example.py)
- [run_phenotype_analysis.py](run_phenotype_analysis.py)

## Usage
```bash
uv run python scripts/phenotype/load_antwiki_example.py --help
```
