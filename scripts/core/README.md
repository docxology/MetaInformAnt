# CORE

## Overview
Command-line helpers for Core workflows. Scripts should remain thin wrappers around `src/metainformant/` implementations and be run from the repository root with `uv`.

## Contents
- [fix_disk_space.sh](fix_disk_space.sh)
- [run_demo.py](run_demo.py)
- [setup_temp_dirs.sh](setup_temp_dirs.sh)

## Usage
```bash
uv run python scripts/core/run_demo.py --help
```
