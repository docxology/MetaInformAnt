# Agent Directives: scripts/simulation

## Role
Simulation scripts for generating synthetic data across all METAINFORMANT modules.

## Contents
Module-specific simulation scripts:
- `simulate_core.py` - Core infrastructure simulations
- `simulate_dna.py` - DNA sequence simulation
- `simulate_ecology.py` - Ecological community simulation
- `simulate_epigenome.py` - Epigenomic data simulation
- `simulate_gwas.py` - GWAS data simulation
- `simulate_information.py` - Information theory simulations
- `simulate_life_events.py` - Life events simulation
- `simulate_math.py` - Mathematical biology simulations
- `simulate_ml.py` - ML dataset simulation
- `simulate_multiomics.py` - Multi-omics simulation
- `simulate_networks.py` - Network simulation
- `simulate_ontology.py` - Ontology simulation
- `simulate_phenotype.py` - Phenotype simulation
- `simulate_protein.py` - Protein data simulation
- `simulate_quality.py` - QC data simulation
- `simulate_rna.py` - RNA-seq simulation
- `simulate_singlecell.py` - Single-cell simulation
- `simulate_visualization.py` - Visualization data simulation

Utilities:
- `run_simulation.py` - Run specific simulations
- `_common.py` - Shared simulation utilities
- `IMPROVEMENTS.md` - Planned improvements

## Purpose
Generate realistic synthetic data for:
- Testing analytical methods
- Benchmarking performance
- Creating examples and tutorials
- Validation with known ground truth
