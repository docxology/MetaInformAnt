# MULTIOMICS

## Overview
Command-line helpers for Multiomics workflows. Scripts should remain thin wrappers around `src/metainformant/` implementations and be run from the repository root with `uv`.

## Contents
- [example_basic_integration.py](example_basic_integration.py)
- [example_cca.py](example_cca.py)
- [example_joint_analysis.py](example_joint_analysis.py)
- [run_multiomics_integration.py](run_multiomics_integration.py)

## Usage
```bash
uv run python scripts/multiomics/example_basic_integration.py --help
```
