# Agent Directives: scripts/life_events

## Role
Life course event analysis workflow scripts.

## Contents
Complete life events analysis pipeline:
- `run_life_events_analysis.py` - Main analysis pipeline
- `generate_synthetic_data.py` - Create test data
- `validate_data.py` - Data validation
- `learn_embeddings.py` - Train event embeddings
- `export_embeddings.py` - Export trained embeddings
- `train_model.py` - Train prediction models
- `predict_outcomes.py` - Run predictions
- `interpret_predictions.py` - Model interpretability
- `analyze_intervention.py` - Intervention analysis
- `compare_groups.py` - Group comparisons
- `generate_visualizations.py` - Create visualizations
- `visualize_sequences.py` - Sequence visualization
- `generate_statistical_summary.py` - Statistical summaries
- `life_course_example.py` - Example workflow

## Workflow Order
1. Generate/validate data
2. Learn embeddings
3. Train model
4. Predict outcomes
5. Interpret and visualize
