# MATH

## Overview
Command-line helpers for Math workflows. Scripts should remain thin wrappers around `src/metainformant/` implementations and be run from the repository root with `uv`.

## Contents
- [run_math_modeling.py](run_math_modeling.py)
- [verify_optimizations.py](verify_optimizations.py)

## Usage
```bash
uv run python scripts/math/run_math_modeling.py --help
```
