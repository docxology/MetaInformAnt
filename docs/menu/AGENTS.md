# Agent Directives: menu

## Role
Documentation agent for the interactive menu and discovery system.

## Module Scope
- Script discovery and categorization from the scripts/ directory
- Interactive terminal menu with breadcrumb navigation
- Script execution with argument prompting and validation
- Menu data structures (MenuItem, Menu, MenuSystem, MenuHistory)

## Key Source Files
- `src/metainformant/menu/core/discovery.py` - Script discovery and metadata extraction
- `src/metainformant/menu/core/executor.py` - Script execution engine
- `src/metainformant/menu/ui/display.py` - Terminal display and formatting
- `src/metainformant/menu/ui/navigation.py` - Navigation logic and data structures
