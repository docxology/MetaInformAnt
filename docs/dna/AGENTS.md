# Agent Directives: docs/dna

## Role
Documentation for DNA sequence analysis, population genetics, and phylogenetics.

## Contents
Documentation covering:
- Sequence processing (FASTA/FASTQ, GC content, k-mers)
- Alignment algorithms (Needleman-Wunsch, Smith-Waterman)
- Population genetics (Tajima's D, nucleotide diversity, FST)
- Phylogenetic analysis (neighbor-joining, UPGMA)
- Variant analysis (VCF parsing, mutation classification)
- NCBI/Entrez integration

## Key Algorithms
- Global/local sequence alignment
- Population summary statistics
- Phylogenetic tree construction
- Neutrality tests
