# Agent Directives: config/archive

## Role
Archived amalgkit configurations for reference and historical purposes.

## Contents
Legacy species configurations that are:
- No longer actively maintained
- Kept for reference/reproducibility
- May need updates for current amalgkit version

## Species Archived
Configurations for various ant species:
- Acromyrmex echinatior
- Apis mellifera (honey bee)
- Atta cephalotes
- Camponotus floridanus
- Harpegnathos saltator
- Lasius neglectus
- Monomorium pharaonis
- Myrmica rubra
- Ooceraea biroi
- Solenopsis invicta
- Temnothorax species
- Vollenhovia emeryi
- Wasmannia auropunctata

## Usage Note
These configs may require updates to work with current code. Prefer configs in `config/amalgkit/` for production use.
